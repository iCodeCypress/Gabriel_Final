HTML stands for HyperText Markup Language
    -the skeloton of the website

CSS stands for Cascading Style Sheet
    -the style of the html (allow you to change the appearence)

Link tag-link 
    -allow linking html and css files together
    -located in the head tag

Body Tag - body
    -the main part of the website
    -this is where we type in our html codes

Header Tag - h1, h2, h3, h4, h5, h6
    -smaller the number, bigger the fonts
    -bolded

Paragraph tag - p
    -small font
    -not bolded

Image tag - Img
        -src stands for source
        -takes in image links for our files

Anchor tag - a
    -href stands for hyperlink reference, 
    - takes in a http link or a file

Divider tag - Div
    - use as a container for a set of html tags

Unordered List - ul
